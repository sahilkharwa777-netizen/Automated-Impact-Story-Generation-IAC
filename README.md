# Automated Impact Story Generation using AI

## Overview

This project generates professional impact stories from participant feedback using AI-based storytelling workflows.

The system collects feedback, anonymizes data, generates stories, enables human review, and stores approved stories in a searchable repository.

## Features

- Participant feedback collection
- Consent management
- Data anonymization
- AI-powered story generation
- Story templates:
  - Social Media Story
  - Blog Story
  - Partner Case Study
- Human review and approval workflow
- SQLite story repository
- Searchable impact story database
- AI ethics and privacy guidelines

## Tech Stack

- Python
- Streamlit
- Pandas
- SQLite
- Transformers
- Scikit-learn
